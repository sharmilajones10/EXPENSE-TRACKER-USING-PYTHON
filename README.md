**Expense Tracker (Python)**

## About the Project
This is a simple **Expense Tracker program built using Python**.  
It reads expense data from a CSV file and helps you understand where your money is going by showing totals, categories, income, and expenses.

This project was created mainly for **learning purposes** and to practice working with real-world data using Pandas.

## What This Program Can Do
- Read expense data from a CSV file  
- Show all transactions  
- Separate income and expenses  
- Calculate total spending and earnings  
- Analyze expenses by category
- 
## Tools Used
- Python  
- Pandas  
- CSV file handling  

## Dataset
The CSV file contains basic transaction details like:
- Date  
- Account  
- Category  
- Note  
- Amount  
- Currency
